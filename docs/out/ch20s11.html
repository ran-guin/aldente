Gel Runs
Create 6 Gel Runs (but only starting 2 of them)

  1. Scan 6 Gel Trays, 6 Gel Combs and a solution that contains Agarose in its "Stock Name"

  2. Comment some of the Gels as desiered

  3. Click on "Create" button to create the Runs. Check the barcode and verify the content and their alignment

  4. Verify the creation of entries in RunBatch, Run and GelRun tables (1 to 1 relation between all)

  5. RunBatch should contain TBD Employee and Equipment. FK_Plate__ID,Run_Directory and Run_DateTime must be NULL

  6. Create a 384-Well Library Plate, Apply an Enzyme to it and aliquote it into 4 96-well Plates.

  7. Optionally set the Growth Status on some of the wells on the 96-well Plates from the Plate homepage. (To see the result when Lanes are created for these wells)

  8. Scan 2 of the printed Gel Run Barcodes + 2 Plates + 1 Gel Box equipment. This will start the run

  9. Check the recently printed barcode, verify the content and their alignment. Make sure it has a proper run directory

  10. Verify the RunBatch.FK_Equipment__ID and FK_Eqmployee__ID are set as well as FK_Plate__ID, Run_Directory and Run_Datetime

  11. Optionally repeate all of the above, but this time add a GelRun attribute entry to the Attribute table

  12. View the Gel Runs just started from the Mapping Summary Page. Confirm all the data appears and they were inputed in the system

  13. Scan the Gel Image (or any Gel Image) and save it in /home/alDente/public/upload/gel_run_data

Bioanalyzer Run
Creating Runs

  • Before you begin:

    • make sure the bioanalyzer is barcoded with equipment type 'Bioanalyzer'

    • you need a test .csv result file (/home/echang/Affy/Bioanalyzer_test.csv) and a test electropherogram image (/home/echang/Affy/Bioanalyzer_test.bmp)

  1. Scan sample tubes and bioanalyzer (e.g.pla76079pla77690pla77691pla78557

    pla79246pla79624equ640)

  2. Under the bioanalyzer run request page, enter some comments and change some dilution factors (try both decimals and fractions)

  3. After creating the run, note the master run ID. Follow the instructions and save the test file and image(s) under /home/sequence/alDente/upload/BioanalyzerRun/New.

  4. Run /opt/alDente/versions/user/bin/update_run.pl (if cron job not set up yet) to process the test files

Viewing Run Data

  1. Scan one of the run ID (e.g. Run46004) of the runs you just created (does not have to be master run ID). Follow the 'Home Page' link to go to the home page with data you just uploaded.

  2. Try viewing the .csv files and make sure the images you uploaded should be displayed and converted to .jpg format.

    Check to make sure the comments and dilution factors are what you have typed in.

  3. Go to the Bioanalyzer 24H page (Gene Expression, Summaries tab, 'Bioanalyzer run search' link), search for the runs you just created (by DateTime, for example). You should be able to view the data you uploaded and the thumbnail images

Spect Run
Creating Runs

  • Before you begin:

    • make sure the spectrophotometer is barcoded with equipment type 'Spectrophotometer'

    • you need a test .xls result file (/home/echang/Affy/Spect_test.xls)

    • you need to create 3 samples (in tubes) in the database; one for a water, one for salmon sperm DNA, and one for human genomic DNA (i.e. add Original Source, Source, Plate, Sample, Plate Sample)

  1. Create a new spect rearray plate: scan a source plate, click 'Wells' tab, click 'Pick Wells to ReArray'

  2. Select proper plate format, size, and library; then select a few source wells (make sure group select works) and click 'ReArray Wells'

  3. Select a few source wells, and double-click on well F1. The selected wells should be filled by column on the target plate starting from F1.

  4. After creating the rearray, note the rearray plate ID. Scan another plate and pick wells to rearray using the existing rearray plate. After selecting source wells and proceeding to the target well selection page, wells with samples from the previous source plate should be shaded grey and disabled. Trasfer new source wells to the plate

  5. When the spect rearray plate is ready, scan the plate and spectrophotometer (e.g.Pla119597Equ641)

  6. Under the spect run request page, select two water wells, two ssDNA wells, and two hgDNA wells. Scan in the tube for each. Select well F1 and click 'Clear Selection'

  7. After creating the run, note the run ID. Follow the instructions and save the test file under /home/sequence/alDente/upload/SpectRun/New.

    Without changing the display configuration, well F1 should not appear on the list of used wells.

  8. Run /opt/alDente/versions/user/bin/update_run.pl (if cron job not set up yet) to process the test files

Viewing Run Data

  1. Scan the run ID (e.g. Run46008) of the run you just created. Follow the 'Home Page' link to go to the home page with data you just uploaded. Data for well F1 should not be displayed.

  2. Try viewing the .xls file.

  3. Go to the Spect 24H page (Gene Expression, Summaries tab, 'Spectrophotometer run search' link), search for the run you just created. You should be able to view the data you uploaded. Data for well F1 should not be displayed.
