1. alDente Public Page

The alDente External Collaborator page is the gateway to alDente LIMS for external users who collaborate with the GSC. The functionalities that are supported on this page are very limited due to security reasons and restrictions. These functionalities are listed in the following section. These collaborators require to have an LDAP/LIMS account on the GSC system in order to be able to access it. For more information please contact aldente@bcgsc.ca Functionalities Supoprted by the alDente LIMS Public Page:

1.1. Submissions

1.1.1. Submit a series of information about a collection of samples that are being sent to the GSC

This information can possibly include

  • Organism information

  • Collaborator informaton

  • Collection information

  • Vector/Enzyme/Primer/Antibiotic information

  • Set of Goals for this collection (ie. 384-well Plates to sequence or Target number of clones to fingerprint)

1.1.2. Submit a series of information about Source Materials that are being sent to GSC

This information can possibly include

  • Ligation

  • Microtiter

  • Library Segments

  • Transformed/Sorted Cells

1.1.3. Submit a work request for a given GSC LIMS Library

This information can possibly include

  • The number of Containers (Plates/Tubes) and their format that are being sent to GSC

  • A set of Goals for this collection (ie. 384-well Plates to sequence or Target number of clones to fingerprint)

  • A set of Primers that are to be used in-case these containers are to be sequenced

1.1.4. Monitoring the state of their previous submissions

Collaborators can constantly watch at what state their pervious submissions are at. Whether they have are pending approval, rejected, or approved. They can also submit any number of files along with their submission using their web browser.
