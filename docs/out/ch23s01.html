1. RGTools Modules

These modules should be usable by anyone, and provide generic functionality that was originally used by the alDente LIMS. Generally only standard perl libraries (if any) are required to allow the use of any routines in these modules.

  • HTML_Table

    This object is an ease-of-use utility for creating tables in HTML

  • Code

    This object represents a Perl source code. It has the ability to insert Perldoc and re-organize a source file.

  • Views

    This facilitates automatic viewing in HTML view..

  • Graph

    This is a wrapper around GD::Graph and its various subclasses

  • Object

    Base class for all alDente modules

  • RGIO

    A collection of miscellaneous utility functions

  • RGmath

    A collection of math utility functions

  • Read

  • Conversion

    A collection of functions for converting volumes, units, dates, etc.

  • Reads

  • TLC

    Tips/Links/Code

  • Directory

    This module auto-documents modules

  • String
