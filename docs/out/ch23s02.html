2. SDB Modules

This directory contains modules specific to SQL database handling. While developed for the alDente LIMS, these modules should not be dependant upon any structure specific to alDente. There are some formatting standards which should be followed for some of these routines to be utilized. These may be found in the alDente documentation or in the module descriptions themselves.

  • DB_IO (deprecated)

    This modules provides general SQL database functionality (non-specific to one database)

  • GSDB (deprecated)

    Generic database functions. Most of its functions have been copied/moved over to DBIO.pm

  • ChromatogramHTML

  • DB_Form

    Generic database entry form for multiple related tables. This is used by most data entry functions in LIMS.

  • Session

    Stores user session

  • DB_Record

    Generic database entry form for single tables.

  • Submission

    This module provides submission_related functionality

  • Issues

    Issue tracker module.

  • DBinput

    Module that handles file (submission) input

  • DB_Form_Viewer

    Handles form input for DB_Form

  • Data Viewer

  • DB_Query (deprecated)

    Module for querying an SQL database.

  • Histogram

    This package allows generation of histograms as png or gif images.

  • Errors

  • CustomSettings

    Custom globals for LIMS SDB modules

  • DB_Object

    Class module for an entity that can span multiple tables

  • Report

    This modules provides general reporting functions

  • DBIO

    Database input/output object.
