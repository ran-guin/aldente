3. Changes for Release 2.20 (Nov / 2004)

3.1. General Changes

  • Solution Handling

    • Dispense function added (no barcode generated)

    • Aliquot units allows for ul,ug etc.

    • General information displayed on homepage

  • Plate/Tube Handling

    • Separate Transfer / Aliquot options

    • Allow Export button

    • Track Aliquot / Transfer volumes (for BOTH Plates and Tubes)

    • Decant option (remove volume from tube / wells)

    • Track Current Volume of Contents

    • Define Sample Alias button

    • Track Empty, Problematic Wells

  • Equipment Handling

    • Add general Info table on homepage

  • Chemistry Calculator

    • Enable automatic initialization of parameters.

    • Enable saving copy as new brew

  • Protocols

    • Tidy up interface for defining protocols

    • Add transfer volume tracking

    • Allow multiple solution tracking (eg. apply different solutions to two halves of plate set)

  • Rearrays

    • Enhancement: Created Rearray homepage - moved all rearray functionality to this page

    • Enhancement: Rebuilt rearray interface for ease of use

    • Enhancement: Can now select rearray operations on multiple rearrays (using checkboxes)

    • Enhancement: Allowed the QPIX to do double-dipped, unordered rearrays

    • Enhancement: Allowed user to abort reserved and on order rearrays (limited capability)

    • Enhancement: Speed boost - added missing index.

    • New feature: remapping primer plates to match rearrays

    • New feature: multiprobe remapping of primer plates

    • New feature: search capability for rearrays

    • New feature: multiprobe rearray capability

    • New feature: rearray summary page

  • Primer Plates

    • New primer type: Amplicon

    • New field: Primer Plate notes

    • Enhancement: Changed direction field to 'Forward' and 'Reverse' (rather than 5 and 3) for less ambiguity

    • Enhancement: Enabled mixed-type primer plate orders (amplicons and sequencing oligos on the same plate)

    • Enhancement: New status - 'To Order'

    • New feature: remapping primer plates to match rearrays (and creating new solution ids for the remapped plates)

    • New feature: Created a central viewing area for primer plates, with minor search capability

    • New feature: Allowed regeneration of primer orders through the LIMS interface

    • New feature: Users can now view the status of ordered custom primer plates

3.2. General Fixes

  • various barcode Printing issues fixed

  • Box numbering correction

  • Rack alias generation adjusted

  • Ensure Box extraction being tracked properly

  • Ensured that permissions will be checked when sample sheets are created

  • Rebuilt multirow-edit form to make it faster and easier to use

3.3. Extra Requests handled

  • spatial calibration issue on 3730-6 resolved

  • Trace submissions generated for Bovine BACs

  • 4 rearrays from WS999 and all WS888 rearrays backfilled

  • New feature: alpha code for receiving TTR samples

3.4. API updates

  • New feature: Added get_Rearray_locations function.

  • New feature: Add_Plate function

  • Enhancement: Expanded order_custom_primer_plate function to accomodate amplicon and mixed plates.

  • Enhancement: expanded capabilities of get_sample_ids

  • Speed up queries using study. (also allow usage of either study_name or study_id).

3.5. Database changes

  • Revising RNA_Source Library structure to simplify and make more powerful

  • New feature: Added SDB-level triggers - can use Perl or straight SQL

  • Maintenance: Redesigned pooling.
