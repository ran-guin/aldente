3. Re-Initializing after a Re-Boot

If you need to perform a full cold reboot, the configuration settings may have to be reset. The following procedure should set the unit back up to operate properly:

  • Start | Settings | Connections | Network

  • Select Spectrum24 Wireless LAN PC Card

    • enter the IP address from the sticker on the back (e.g. 10.1.1.152 for ppt02) - NOT the IP address from the bridge (10.1.1.150)

    • subnet mask is 255.255.255.0

    • default gateway is 10.1.1.1

    • hit OK in top right of screen
