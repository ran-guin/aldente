3. Primary Class Definitions for alDente

3.1. Laboratory Object Classes

  • Sample - (eg Plate)

    A specific type of laboratory sample object - associated with various actions

    • Transfer

    • Move

    • Set Unused Wells

    • Set No Grows

    • Fail

    • Throw Away

    Information extracted from this object may include:

    • View Preparation History

    • Show Applied Solutions

    • View Ancestry

  • Solution

    A reagent or solution in stock, associated with various actions:

    • Mix

    • Open

    • Use (specified quantity)

    • Throw Away

    Data extracted from this object may include:

    • Original reagents : list of original reagents (& quantities) used to make solutions.

    • Used : list of instances when used (& quantities)

  • Equipment

    A piece of equipment in stock, associated with various action

    • Apply Maintenance Process

    • De-commission

    Data extracted from this object may include:

    • Maintenance History

    • Warranty summary

    • Service History

    • Used : list of instances when used

3.2. Laboratory Process Classes

  • Preparation

    An object that enables loading of protocol steps and storing of Preparation steps

  • Protocol

    An object that enables definition and extraction of customized Lab_Protocols.
