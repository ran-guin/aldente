2. Control_Plates

2.1. Using Control Plates and Comparing Plates prepared in parallel

From the Last 24 Hours view, there is a 'QC' link that allows users to view the data results for all plates prepared concurrently with the given plate. It also specifically looks for plates that are marked as Control Plates (given an attribute '*_Control_Plate' which references a control plate).

Figure 16.1. 

Clicking on the 'QC' link to the right of the run maps takes the user to an overview of results for related control plates (if applicable) and other plates handled simultaneously in the lab:

Figure 16.2. 

Users may also expand the given folders to see the explicit plates handled together with the current one:
