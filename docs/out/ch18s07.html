7. Adding a New Department

7.1. Adding a New Department

  1. Copy Lab_Department.pm to 'new_deptr'>_Department.pm

  2. replace applicable occurances of 'Lab' to 'new_dept')

  3. Insert new Department 'new_dept' (via LIMS_Admin page)

  4. Add groups as applicable for this new department (add at least 1)

  5. Add at least one employee to this group. (will not show up until at least 1 active user)

  6. Change table permissions if necessary

  7. Refresh Parameters

7.1.1. Adjusting group privileges

  • Adjust employee group privileges using LIMS Admin page

  • Set Department ID for Employee by using the Edit interface
