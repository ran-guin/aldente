5. Bioanalyzer Runs

In order to run a bioanalyzer, the sample must be transferred to a bioanalyzer chip and then loaded onto a bioanalyzer scanner.

5.1. Transferring samples to a bioanalyzer chip and initiating a bioanalyzer run

  • Scan the tubes that contains the samples (barcode prefix PLA) (up to 12 samples for each run) in order (first scanned sample goes to first well), then scan the bioanalyzer scanner equipment (barcode prefix EQU).

  • A page will be displayed allowing the user to enter the sample dilution (1/2, 0.5, etc.) (must be smaller or equal to 1) and comments (optional) for each sample.

    Figure 13.6. Bioanalyzer Run Request

Each sample will create a bioanalyzer run, but to make runs easier to track, all the runs from the same bioanalyzer chip are identified by a master run ID, which is the same as the run ID for the first sample on the chip. The LIMS will print out a barcode for this run ID.

Figure 13.7. Bioanalyzer Run Request - Result

5.2. Saving data in LIMS

After scanning, the bioanalyzer machine generates an .xad data file that includes texts and images. In order to save these information in LIMS, the following needs to be done:

  • Convert the .xad file to a .csv file (save as <master run ID>.csv).

  • Convert the electropherogram into .bmp images (save as <master run ID>_<well #>.bmp). For example, 43034_3.bmp is the image for the third sample.

  • Save the .csv file and images in a designated directory.

LIMS will obtain these files every 5 minutes and save the information into databases.

5.3. Viewing data in LIMS

There are two ways to view the bioanalyzer data in LIMS:

  • Bioanalyzer homepage - the following are a few ways a user can get to the bioanalyzer homepage:

    • scan in the run ID and follow the 'Home Page' link, if the user knows the run ID.

    • scan in the sample tube (barcode prefix PLA), if the user wants to see all data associated with the sample. Under the 'Data' tab, all the runs associated with the sample are displayed. Follow the links to view the data for each run.

      Figure 13.8. Bioanalyzer Run - View data

  • Bioanalyzer Summary Page - allow user to search for bioanalyzer runs:

    • From the lab home page, click on the 'Summaries' tab, and follow the 'Bioanalyzer run search' link.

5.4. Tracked data

The bioanalyzer run information stored are as follows:

  • Scanner Equipment. the bioanalyzer machine used for scanning

  • Dilution Factor. the sample dilution factor for the sample (must be smaller or equal to 1) (e.g. 0.5 means 1 in 2 dilution)

  • Well. well number on the bioanalyzer chip (1 to 12)

  • Well Status. status of the well ('Empty','OK','Problematic','Unused')

  • Well Category. category of the well ('Sample','Ladder')

  • RNA Concentration. RNA concentration of the sample loaded into the bioanalyzer chip well; this does not take dilution factor into account

  • RNA Concentration Unit. RNA concentration unit

  • RNA Integrity Number. RNA Integrity Number, used for assessing RNA sample quality

  • Read Error. error from a well's reading ('low concentration','low RNA Integrity Number')

  • Read Warning. warning from a well's reading ('low concentration','low RNA Integrity Number')

  • Sample Comment. comment for each well's sample

  • File Name. the file name of the .xad file generated from the bioanalyzer machine
